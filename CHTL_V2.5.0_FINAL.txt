                        script 块支持
═══════════════════════════════════════════════════════════════════════════

✅ 纯 JavaScript        - 完整保留
✅ 纯 CHTL JS          - 完整转换  
✅ 混合使用             - 完美共存（推荐）

示例：
```javascript
script {
    // 1. JavaScript 部分 - 数据和逻辑
    const state = { count: 0 };
    function increment() { state.count++; }
    
    // 2. CHTL JS 部分 - DOM 选择
    const counter = {{#counter}};
    const btn = {{.btn}};
    
    // 3. 混合部分 - 事件处理
    btn &-> click: () => {
        increment();  // JS 函数
        counter->textContent = state.count;  // CHTL JS
    };
    
    // 4. JavaScript 部分 - 异步
    fetch('/api/data')
        .then(res => res.json())
        .then(data => {
            {{#result}}->innerHTML = data;  // CHTL JS
        });
}
```
