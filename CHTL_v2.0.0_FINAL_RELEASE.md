## 🚀 Getting Started

### Installation
```bash
git clone https://github.com/chtl-lang/chtl
cd chtl
python3 build.py configure
python3 build.py build
```

### First Program
```chtl
[use html5]

div {
    class: hello-world;
    h1 { text: "Hello, CHTL v2.0!"; }
}
```

### Compile
```bash
./build/chtl your-file.chtl -o output.html
```
